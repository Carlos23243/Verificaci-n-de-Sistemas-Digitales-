# Verificaci-n-de-Sistemas-Digitales-
# Diseño y Verificación de Sistemas Digitales en VHDL
Este repositorio contiene el desarrollo práctico de la Unidad 4 para la asignatura de Fundamentos de Sistemas Digitales. El objetivo principal es el modelado, simulación y validación de componentes electrónicos utilizando el lenguaje de descripción de hardware VHDL.
# Contenido del Proyecto
El proyecto está dividido en cuatro módulos fundamentales que cubren desde la lógica básica hasta la verificación funcional:

**Parte A:** Estructura básica de VHDL (Compuerta AND).

**Parte B:** Manejo de tipos de datos y aritmética (Sumador de 2 bits).

**Parte C:** Componentes combinacionales (Multiplexor 2:1, Decodificadores y Comparadores).

**Parte D:** Simulación y Verificación (Testbenches con generación de estímulos y asserts).
